# Ordasoy DAO Single-Page Web Application

## Overview
A single-page web application for Ordasoy DAO that provides a comprehensive platform for plantation investment, NFT launches, and decentralized governance. The application simulates Solana wallet connection and includes multi-language documentation with PDF export capabilities.

## Core Sections

### Home Section
- Project overview and mission statement
- Problems solved by the platform
- Benefits for founders and investors
- Key statistics and metrics
- Call-to-action elements for getting started

### Plantation Company Section
- Document submission interface for plantation companies
- NFT launch target setting and configuration
- Project timeline management and display
- DAO voting interface for company proposals
- Company updates and news feed
- Provider search functionality
- Investor chat system for communication

### Investor Section
- NFT holdings dashboard per company
- Portfolio overview and performance metrics
- Access to individual company pages
- Investment history and tracking
- Voting power and governance participation

### Launch Section
- Active NFT launches display
- NFT minting interface with quantity selection
- Claiming functionality for purchased NFTs
- Refunding system for failed launches
- Launch status tracking and notifications

## Wallet Integration
- Generic Solana wallet connection simulation
- Wallet connection status indicators
- Mock wallet address display
- Simulated transaction confirmations
- Connection/disconnection functionality

## Documentation System
- Professional Markdown documentation in three languages:
  - English (primary)
  - Russian
  - Kazakh
- Improved project description with proper formatting
- Language switcher for documentation sections
- PDF export functionality with conversion script
- Clear instructions for Markdown to PDF conversion

## Technical Requirements

### Frontend Implementation
- Single-page application with smooth section navigation
- All functionality runs entirely in frontend
- No backend data persistence required
- Mock data for all operations and interactions
- Responsive design for all device types

### Data Management
- Frontend-only state management
- Local storage for user preferences and session data
- Simulated real-time updates for launches and voting
- Mock data for companies, NFTs, and investor portfolios

### User Interface
- Elegant and professional design
- Consistent branding using provided assets
- Intuitive navigation between sections
- Clear visual hierarchy and information architecture
- Accessible design following web standards

### Multi-language Support
- Application interface in English
- Documentation available in English, Russian, and Kazakh
- Proper text encoding for all character sets
- Language selection for documentation viewing

### Export Functionality
- Generate PDF from improved Markdown documentation
- Maintain professional formatting in PDF output
- Support for multi-language PDF generation
- Automated conversion script or clear manual instructions
